# Task

Build a [fomo.family](https://fomo.family)-like **landing page** (minimum) + **trading page**.

## Requirements

1. **Use the ChadWallet brand**
   - Assets: https://drive.google.com/drive/folders/1j4PZng-sJHxqAATUF1WYw1_jm8nyQwCE
   - Mobile app links:
     - Android: https://play.google.com/store/apps/details?id=xyz.chadwallet.www
     - iPhone: https://apps.apple.com/us/app/chadwallet/id6757367474
2. **Sign in** with Apple/Google through Privy.
3. **Supports Solana.**
4. **Rotating token banners** at the top and bottom.
   - Tapping a token in the banner opens its trading page.
5. **Trading UI** *(optional — OK to skip if short on time; can be a follow-up)*
   - **Left:** trending tokens list
   - **Middle:** token info, price chart, list of holders, and live trades
   - **Right:** buy & sell, user's position

## Tools

1. AI tools
2. Next.js, Tailwind
3. Supabase, Cloudflare, Vercel
4. Privy — https://privy.io
